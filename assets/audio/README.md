|---|---|

## Notes

- All LibriVox recordings are **public domain** and free to use.
- For collections, you may need to cut the relevant track using a free tool like **Audacity** or **mp3DirectCut**.
- If an audio file is missing, the app will show a warning in the browser console but will **not crash** — the story still works in Read and Practice modes.
- Recommended format: MP3, 128kbps, mono is fine for voice recordings.
